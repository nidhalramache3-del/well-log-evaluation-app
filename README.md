# Well Log Evaluation & Hydrocarbon Reserves Calculation

A professional Streamlit web application for petrophysical analysis and reservoir volumetrics.

## Features
- **Data Import**: Supports LAS and CSV formats.
- **Petrophysics**: Automated calculation of Vshale, Porosity, and Water Saturation (Archie).
- **Visualization**: Interactive 2D log plots and 3D wellbore geological models using Plotly.
- **Volumetrics**: IOIP (Initial Oil In Place) calculator.
- **Multilingual**: Supports English and French.

## Installation
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   streamlit run app.py
   ```
